# CLion Color Theme for VSCodes
This is a lightweight color theme based on default CLion IDE.

## Supported languages
 - C
 - C++
 - HTML
 - CSS
 - JavaScript

To get the full theme experience set the following settings in VSCode:     
```
"editor.fontFamily": "'Consolas'",
"editor.fontSize": 13
```
